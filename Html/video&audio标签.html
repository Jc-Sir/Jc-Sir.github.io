# video&audio标签

# video标签

# 属性

属性 值 描述
autoplay autoplay 视频就绪自动播放（谷歌浏览器需要添加muted来解决自动播放问题）
controls controls 显示播放控件
width pixels(像素) 播放器宽度
height pixels(像素) 播放器高度
loop loop 播放完后是否循环播放
preload auto(预先加载)、none(不预加载视频) 如果有了autoplay属性就忽略该属性
src 视频地址 视频地址
poster imgurl 加载等待的图片画面
muted muted 静音播放

# audio标签

属性 值 描述
autoplay autoplay 音频就绪自动播放
controls controls 显示播放控件，比如播放按钮
loop loop 播放完后是否循环播放
src 音频地址 音频地址
